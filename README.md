Проект - Игра

Команда - TRS (The running sloth)

Участники команды:
    Шалятов Роман (лидер)
    Фролов Арсений
    Дружинин Алексей

Описание игры:
    TRS - пиксельный рогалик, который берёт свою идею из уже существующей игры “Soul Knight”.
    Приключения, сражения, интересные персонажи - вот, что примерно описывает нашу игру.

Куратор - Шуплецов Б. (Яндекс Лицей "Основы промышленного программирования | Д25")


Назначения клавиш:
    На стратовом экране:
        - - Для снижения громкости фоновой музыки на 10%
        +/= - Для повышения громкости фоновой музыки на 10%
        M - отключить/включить музыку
        ESC - для закрытия игры
        F11 - отключить/включить фуллскрин
    На экнаре лобби:
        - - Для снижения громкости фоновой музыки на 10%
        +/= - Для повышения громкости фоновой музыки на 10%
        M - отключить/включить музыку
        ESC - для возврата на начальный экран
        F11 - отключить/включить фуллскрин
    Во время игры:
        - - Для снижения громкости фоновой музыки на 10%
        +/= - Для повышения громкости фоновой музыки на 10%
        M - отключить/включить музыку
        ESC - для возврата в лобби
        SPACE - для паузы игры
        F11 - отключить/включить фуллскрин
        E - активировать портал на новый уровень (1)
        ЛКМ - атака персонажа
        WASD - стандартное управление
        Админ функции:
            Q - снять игровому персонажу 1 хп
            R - сразу переключиться на следующий уровень
            Z - включить режим бессмертия (отсутствует урон от монстров)

1 - Чтобы активировать портал на новый уровень, требуется собрать все сундуки на уровне
